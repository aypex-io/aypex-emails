# Aypex::Emails

## Installation

Add this line to your existing Aypex application's Gemfile:
```ruby
gem "aypex-emails"
```

And then execute:
```bash
bundle
```

And then run the Aypex::Emails install command:
```bash
bin/rails g aypex:emails:install
```

## Testing

```bash
bundle exec rake test_app
```

```bash
bundle exec rspec spec
```
